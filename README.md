# ETH-DAPP-Token-Exchange

## Step-1 
#### Download Meta Mask Extension from Google chrome extension and Sign Up or Login

## Step-2 
#### If you have not any ether in your account then download Ganache from this link https://www.trufflesuite.com/ganache 

## Step-3 Click on QuickStart Ethereum

<img src="https://user-images.githubusercontent.com/68073132/115140364-855b9200-a054-11eb-8a61-a64b3c5251cc.png" width="850" height="600">

#### Copy First Accounts Private Key and Go to Meta Mask-> Import Account -> Paste private key -> Import

<img src="https://user-images.githubusercontent.com/68073132/115140467-1894c780-a055-11eb-9dcf-7cc89b72215e.png" height="550">

## Step-4 
#### Now For Exchanging Ethereum Token to DAPP Token Click on Buy and write amount and Click on  Swap Button

<img src="https://user-images.githubusercontent.com/68073132/115140765-96a59e00-a056-11eb-8ccb-d66c78ec8541.png" height="550">


#### One MetaMask Notification Window Popup
#### Click On Confirm
#### And Reload Page 

<img src="https://user-images.githubusercontent.com/68073132/115140941-6c081500-a057-11eb-8d7b-af0ffc2e4c7a.png" height="550">

Now Check Both Token Balance. 
You can see Notification from meta mask. 
Succefully transfer token from ether to DAPP.

<img src="https://user-images.githubusercontent.com/68073132/115141049-fa7c9680-a057-11eb-81c5-d17c947068d8.png" width="850" height="600">

## Step-5 
#### Now for transfering DAPP token to Ether click on Sell Button
#### And Write amount and Swap it !!!

<img src="https://user-images.githubusercontent.com/68073132/115141286-0026ac00-a059-11eb-9928-2b966ad4f02e.png" width="850" height="600">

#### Now MetaMask Notification Window Popup
#### Click On Confirm 
#### And Reload the page

<img src="https://user-images.githubusercontent.com/68073132/115141395-9bb81c80-a059-11eb-9e53-08ea9c2f68f3.png" height="550">


#### Check Token Balance. Its Updated  
#### You can see Notification from meta mask. 
#### Succefully transfer token from ether to DAPP.
